# Address Extraction from Text

## Prequsites
- Python

## Installation
1. Install libraries
```bash
pip install -r requirements.txt
```

## How to develop
1. Work on Jupyter Notebook
2. Write the streamlit script on Jupyter Notebook
3. Export the notebook to a Python script
4. Run Streamlit 

## How to run
1. Run the script
```bash
streamlit run <script>.py
```
